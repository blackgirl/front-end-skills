СТАНДАРТЫ

Стандарты W3C определяют Открытую Web-платформу для разработки приложений, которая имеет беспрецедентный потенциал и позволяет разработчикам создавать функционально насыщенные интерактивные системы, использующие ресурсы крупных хранилищ данных и доступные на любом устройстве. Хотя границы этой платформы продолжают расширяться, лидеры отрасли практически в один голос говорят о том, как HTML5 станет ее краеугольным камнем. Однако, полная мощь рассматриваемой платформы зависит еще и от многих других технологий, создаваемых консорциумом W3C и его партнерами, в том числе CSS, SVG, WOFF, Семантический Web, XML, а также различные API.

W3C разрабатывает эти технические спецификации и методические рекомендации в рамках процесса, спректированного для достижения максимального консенсуса относительно содержания технического отчета, обеспечения высокого технического и редакционного качества, а также приобретения поддержки со стороны более широкого сообщества.

Если вы изучаете Web-технологии, то можете начать с приведенной ниже вводной части и пройти по ссылкам для получения более подробных сведений.

  •  Web-дизайн и приложения

    Раздел «Web-дизайн и приложения» включает стандарты для создания и рендеринга Web-страниц, в том числе HTML5, CSS, SVG, Ajax и другие технологии для Web-приложений («WebApps»). В этом разделе также содержится информация о том, как сделать страницы доступными для людей с ограниченными возможностями (WCAG), интернационализированными и пригодными для работы на мобильных устройствах.

  •  Web-архитектура

    Web-архитектура базируется на фундаментальных технологиях и принципах, составляющих опору Всемирной паутины, включая URI и HTTP.

  •  Семантический Web

    В дополнение к классическому «Web´у документов», W3C содействует созданию набора технологий для «Web´а данных”, построенного по аналогии с общеизвестными базами данных. Конечная цель Web´а данных заключается в том, чтобы заставить компьютеры делать больше полезной работы и разработать системы, которые могут поддерживать защищенные взаимодействия по сети. Термин «Семантический Web» выражает то, как W3C видит Web связанных данных. Технологии Семантического Web´а позволяют людям создавать хранилища данных во Всемирной сети, составлять словари и писать правила обработки данных. Связанные данные поддерживаются такими технологиями, как RDF, SPARQL, OWL и SKOS.

  •  Технология XML

    XML-технологии, включая XML, XQuery, XML Schema, XSLT, XSL-FO, Efficient XML Interchange (EXI) и другие связанные с ними стандарты.

  •  Web услуги

    Web услуги – это основанная на обмене сообщений технология, часто используемая в Web и в корпоративном программном обеспечении. Web услуги основаны на таких технологиях, как HTTP, XML, SOAP, WSDL, SPARQL и другие.

  •  Web устройства

    Web устройства базируются на технологиях, обеспечивающих доступ к Web повсюду, в любое время и с помощью любого устройства. Это понятие включает в себя доступ к Web с мобильных телефонов и других мобильных устройств, а также использование Web-технологий в потребительской электронике, принтерах, интерактивном телевидении и даже автомобилях.

  •  Браузеры и инструментальные авторские средства

    Web-агенты предназначены для обслуживания пользователей. В этом разделе вы найдете информацию, полезную при разработке браузеров и инструментальных авторских средств, а также поисковых ботов, агрегаторов и механизмов логических выводов.
